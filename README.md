# Pomodoro Timer Web App
 
A simple Pomodoro Timer application designed to improve focus and productivity using structured work and break intervals.

## Features
 
- 25-minute Pomodoro focus timer
- Short break timer
- Start, pause, and reset functionality
- Clean and responsive UI
- Simple and distraction-free design

## Installation & Setup
 
```bash
git clone https://github.com/VarshaVardhiniM/promodoro-timer.git
cd promodoro-timer
npm install
npm start
```
